## ⚙️ Como Rodar

### Pré-requisitos

- Node.js
- Expo CLI ou React Native CLI
- Android Studio ou emulador iOS
- Conta na Expo (se aplicável)

### Passos

```bash
# Clone o repositório
git clone [https://github.com/usuario/projeto.git](https://github.com/dudabrigidio/Autottu.git)

# Instale as dependências
npm install

# Inicie o projeto
npx expo start
npm run web
# ou
npm run android
